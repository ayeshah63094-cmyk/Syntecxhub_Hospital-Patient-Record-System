# Hospital Patient Record System (MySQL)

## Project Overview
The Hospital Patient Record System is a database-based project developed using MySQL.
The main objective of this project is to manage patient records in a hospital environment
in a simple, efficient, and secure manner.

This system allows adding new patients, viewing patient details, updating medical history,
and deleting patient records using SQL queries.

---

## Objectives
- To store patient information in a structured database
- To perform CRUD operations using MySQL
- To manage patient medical history efficiently
- To ensure data privacy and secure access

---

## Technologies Used
- MySQL
- MySQL Workbench
- Windows Operating System

---

## Database Details
Database Name: hospital_db  
Table Name: patients

---

## Table Structure
The patients table contains the following fields:

- patient_id (INT, Primary Key)
- name (VARCHAR)
- age (INT)
- gender (VARCHAR)
- medical_history (TEXT)

---

## Functionalities Implemented

1. Create Database  
   A database named hospital_db is created to store hospital patient records.

2. Create Table  
   A patients table is created to store patient details.

3. Insert Patient Records  
   New patient details are inserted into the database.

4. View Patient Details  
   Patient information is retrieved using SELECT queries.

5. Update Medical History  
   Existing patient medical history is updated using UPDATE queries.

6. Delete Patient Records  
   Patient records are deleted when required using DELETE queries.

---

## Data Privacy and Security
- Patient data is stored securely in the MySQL database.
- Access to the database is protected using MySQL username and password.
- Only authorized users can access and modify patient records.
